# Portfolio-Template# Portfolio Template 🚀

A modern, futuristic, and fully customizable portfolio website built using pure HTML, CSS, and JavaScript.

This template is designed for students, developers, researchers, freelancers, and professionals from **any stream or domain** — not just Machine Learning.

---

## ✨ Features

- Modern Glassmorphism UI
- Fully Responsive Design
- Animated Hero Section
- Interactive Project Cards
- Smooth Scroll Animations
- Dynamic Portfolio Content
- Skills Showcase
- Dark/Light Theme Toggle
- Particle Background Effects
- Typed Text Animation
- Mobile Friendly Layout
- Easy Customization

---

## 📂 Project Structure

You can use this project in **two ways**:

### Option 1 — Single File Setup

Keep everything inside one `index.html` file:
- HTML
- CSS
- JavaScript

This is best for:
- Beginners
- Quick editing
- Easy deployment
- Simple customization
